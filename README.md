A **dynamic XR Engine** to help you stay in the **flow**

---

This is a simple example project which you can use to figure out how to work
with WasserXR.

## Usage

To run the example run

```bash
git clone git@github.com:LarsZauberer/WasserXR-Demo.git
cd WasserXR-Demo
```

then run it with

```bash
cargo run --release --features static
```
